== Installation ==

Installation process of this plugin is very easy. First one has to download our Hover Effects Ultimate plugin from Wordpress Directory.

<strong> Image Hover Effects Ultimate</strong> WordPress plugin is very user friendly. You can install it at many ways. Here bellow we provide two different way of <strong>installation</strong>:

<blockquote>

<ul>
<strong>Option 1</strong>
<li>Download the plugin image-hover-effects-ultimate.zip from WordPress Plugin Directory.</li>
<li>Unzip the zip file and extract the plugin folder named <strong>image-hover-effects-ultimate.zip</strong></li>
<li>Left that plugin folder into your wp-content/plugins folder.</li>
<li>Go and refresh your Wordpress administration panels and from the menu bar click on Plugins.</li>
<li>Now you may see your plug-in listed under inactive plug-in tab.</li>
<li>Click active to turn your wordpress plugin on.</li> 

</ul>

</blockquote>

<blockquote>

<strong>Option 2</strong>
<ul>
<li>Like the first option you have to download the zip file from download location.</li>
<li>Unzip the zip file and extract the Image Hover Effects Ultimate plugin folder.</li>
<li>Double click on the installable folder in order to get image-hover-effects-ultimate.zip.</li>
<li>Go to wordpress admin panel and press Add New from the Menu bar under plugins tab.</li>
<li>Select upload link and drug the available image Hover Effects Plugin file and click install now.</li>
<li>After successfully installed finally click on ‘activate’ to turn on.</li>
</ul>

</blockquote>

That’s all about the installation process. For further query you’re welcome to contact us.
